# Adaptive File System Honeypot Agent

My final year diploma project.

## What It Does

- Monitors file system activity
- Detects suspicious behavior
- Creates decoy files when threats detected
- Catches attackers accessing decoys
- Sends alerts

## Progress

- [x] Day 1: Project setup
- [ ] Day 2: File monitoring
- [ ] Day 3: Threat detection
... (more to come)

## How to Run

```bash
# Activate virtual environment
source venv/bin/activate  # or venv\Scripts\activate on Windows

# Run agent
python src/agent/main.py
```
